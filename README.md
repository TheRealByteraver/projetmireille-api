# Rest API

This is a REST API written using the Express framework in conjunction with Sequelize as ORM.

# Technology Stack

Node.Js (Express + Sequelize)

# run on server

- in the root dir, run `npm run build`
- start with `pm2 start dist/app.js`
- or reload with `pm2 reload`
- diagnostics: `pm2 logs my-api`
